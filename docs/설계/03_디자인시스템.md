- **본문 텍스트**: slate-700 (#334155)
- **헤딩**: brand-900 / slate-900
- **보조/캡션**: slate-500
- **배경**: white / slate-50
- **경계선**: slate-200

> 클론 시: 신규 사업 사이트는 `tailwind.config`의 `brand` 스케일과 `site.ts`의 컬러만 교체하면 전체 톤이 바뀐다.

## 2. 타이포그래피

- **한글 본문/헤딩**: `Pretendard` (웹폰트, 가독성·기업 신뢰감 우수)
- **라틴/숫자**: `Inter` 폴백
- 폰트 스택: `Pretendard, Inter, -apple-system, "Segoe UI", "Apple SD Gothic Neo", sans-serif`

| 역할 | 크기(데스크탑) | 굵기 |
|------|----------------|------|
| H1 / Hero | 40–56px (text-5xl/6xl) | 700 |
| H2 섹션 | 30–36px (text-3xl/4xl) | 700 |
| H3 | 20–24px (text-xl/2xl) | 600 |
| 본문 | 16–18px (text-base/lg) | 400 |
| 캡션 | 14px (text-sm) | 400 |

---

## 3. 레이아웃 & 스페이싱

- 컨테이너 최대 폭: `max-w-7xl` (1280px), 좌우 패딩 `px-4 sm:px-6 lg:px-8`
- 섹션 세로 여백: `py-16 md:py-24`
- 그리드: 카드 1(모바일)/2(태블릿)/3~4(데스크탑) 열
- 반응형 브레이크포인트: Tailwind 기본(sm 640 / md 768 / lg 1024 / xl 1280)

---

## 4. 컴포넌트 규격

| 컴포넌트 | 설명 |
|----------|------|
| Header | 로고 + 글로벌 내비 + 문의 CTA, 스크롤 시 그림자, 모바일 햄버거 |
| Footer | 회사정보/연락처/메뉴/카피라이트 (site.ts 출력) |
| Hero | 풀폭 그라데이션(brand) 배경 + 슬로건 + CTA |
| SectionHeading | 라벨 + 제목 + 설명 (섹션 상단 공통) |
| Card | 아이콘/제목/요약 + 링크 (서비스·사업영역 공통) |
| Button | variant: primary(brand-500) / outline / ghost |
| CTASection | 문의 유도 배너 |
| ContactForm | 입력 필드 + 제출 버튼 |
| Badge/Pill | 카테고리·태그 표시 |

### 버튼 스타일
- primary: `bg-brand-500 text-white hover:bg-brand-600 rounded-lg px-5 py-3`
- outline: `border border-brand-500 text-brand-500 hover:bg-brand-50`

---

## 5. 비주얼 톤 & 모션

- 카드/이미지: 부드러운 라운드(`rounded-xl`), 옅은 그림자(`shadow-sm` → hover `shadow-md`)
- 모션: 과하지 않게 — fade/슬라이드업 진입, 호버 트랜지션 150–200ms
- 아이콘: 라인 스타일(lucide 계열) 통일
- 이미지: 산업현장/자동화/기술 느낌, 블루 오버레이로 톤 통일 (v1은 placeholder)

---

## 6. 접근성

- 색 대비 WCAG AA 이상(브랜드 블루 위 흰 텍스트 확인)
- 모든 인터랙티브 요소 포커스 링
- 이미지 alt, 시맨틱 태그, 폼 label 연결
