## 2. 목표

1. **신뢰감 있는 기업 이미지 전달** — 산업 자동화·스마트팩토리·AI 전문 기업으로서의 전문성.
2. **문의 전환** — 잠재 고객이 쉽게 연락할 수 있는 명확한 CTA와 문의 폼.
3. **콘텐츠 확장성** — 제품/서비스, 소식을 데이터로 추가 가능.
4. **사이트 클론 확장성** — 향후 연관 없는 신규 사업을 위한 별도 사이트를 쉽게 복제·재브랜딩 가능. (핵심 제약 → [04_아키텍처.md](04_아키텍처.md) 참조)

## 4. 범위 (Scope)

### 4.1 포함 (In-Scope) — v1
- 홈(메인), 회사소개, 제품·서비스, 블로그/뉴스, 문의 페이지
- 반응형(모바일/태블릿/데스크탑) 디자인
- 블루 계열 "테크 모던" 디자인 시스템
- 문의 폼 UI (전송 백엔드는 단계적 — 우선 mailto/외부 폼 서비스 연동 옵션)
- SEO 기본(메타태그, OG, sitemap, robots)
- i18n 구조(한국어 콘텐츠 우선, 영어 확장 대비)
- 브랜드/콘텐츠 중앙 설정으로 클론 용이성 확보

### 4.2 제외 (Out-of-Scope) — v1
- 결제/이커머스
- 회원 가입/로그인
- 영문 콘텐츠 실제 작성 (구조만 마련)
- CMS 관리자 화면 (콘텐츠는 코드/마크다운 기반)

## 6. 비기능 요구사항

- **성능**: 정적 생성(SSG) 기반, Lighthouse 성능 90+ 목표.
- **유지보수**: 컴포넌트·상수 모듈화(CLAUDE.md 코드 작성 규칙 준수).
- **확장성**: 사이트 클론 → 설정/콘텐츠 교체만으로 신규 사업 사이트 생성.
- **호환성**: 최신 Chrome/Safari/Edge, 모바일 Safari/Chrome.
