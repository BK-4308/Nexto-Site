>
> ✅ **구현 반영(2026-07-03)**: 홈 '핵심 사업'과 일관성을 위해 상세 페이지는 **4종**(산업 자동화 / MES / 스마트 팩토리 / AI)으로 정리했다.

### S1. 스마트 팩토리 구축 (MES·생산관리)
- **요약**: 흩어진 제조 현장 데이터를 하나로 연결해, 보이지 않던 생산을 실시간으로 보이게 만듭니다.
- **문제 정의**: 설비·작업·품질 데이터가 수기·엑셀로 분산되어 실시간 파악이 어렵고, 문제 발생 후에야 대응하게 됩니다.
- **넥스투 접근**: 현장 진단 → 데이터 표준화 → MES/생산관리 시스템 설계·구축 → 대시보드로 실시간 가시화.
- **주요 범위**: 생산실적 집계 · 설비 모니터링 · 품질/불량 추적 · 작업지시/진척 관리 · 리포트 자동화.
- **기대 효과**: 생산 가시성 확보, 불량 원인 추적 시간 단축, 데이터 기반 의사결정.
- **관련 사례**: `[사례 확인 필요]`
- `[IMG: MES 대시보드 / 공장 모니터링 화면]`

### S2. 산업 자동화 / 설비 자동화(FA)
- **요약**: 반복·위험 공정을 자동화 라인으로 전환해 생산성과 안전성을 동시에 높입니다.
- **문제 정의**: 수작업 의존 공정은 편차·병목·안전 리스크가 크고 인력 확보도 갈수록 어려워집니다.
- **넥스투 접근**: 공정 분석 → 자동화 설계(설비/제어/센서) → 구축·시운전 → 안정화.
- **주요 범위**: 자동화 라인 설계 · 제어(PLC) · 센서/비전 연동 · 설비 인터페이스 · 라인 통합.
- **기대 효과**: 생산성 향상, 품질 편차 감소, 현장 안전 강화.
- **관련 사례**: `[사례 확인 필요]`
- `[IMG: 로봇 암 / 자동화 컨베이어 라인]`

### S3. AI 공정·품질 최적화
- **요약**: 자동화에 AI를 더해 이상을 미리 감지하고 품질을 예측하는 '스스로 판단하는 공장'으로.

- **주요 범위**: 설비 이상 감지 · 예지보전 · 품질 예측 · 공정 파라미터 최적화.
- **기대 효과**: 비계획 정지 감소, 불량률 개선, 숙련 노하우의 데이터화.
